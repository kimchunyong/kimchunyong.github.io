null 과 undefined
두 타입은 모두 값이 비어있음을 나타낸다.
하지만 미묘한 차이가있음

undefined는 값이 아예 할당되지 않은상태(hoisting에서 볼수있음)
따라서 변수자체에 값이 아예 할당되지 않은 상태이다

null은 개발자가 명시적으로 값이 비어있음을 나타낼때 사용하는값.
