문자열, 숫자, 불린 같은 경우 (deep copy)

예제코드

배열, 일반 객체, 함수 (shallow copy)

변수는 여러개 일수있지만 가르키는 주소는 한개.

배열을 깊은복사를 하기위해선 call함수를 사용
